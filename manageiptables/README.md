# iptrules.sh

Reload iptables rules and backup previous state.

    All files with rules placed in /etc/iptables/conf.d/ .
    All rules-files to load have to be named with ".conf" extension.
    
    All previous states placed in /etc/iptables/prev/

    All example rules-files placed in /etc/iptables/rules/

    All variables placed in /etc/iptables/vars
